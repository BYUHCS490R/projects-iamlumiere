2. What is the root? What is the relationship between folder and file names and the path?

The root is the primary link of your website without any path. One example would be "http://127.0.0.1:3000/". Everything starts in your root which is the starting path. Folder and file names are part of the path.

3. What is a default document?

A default document is the primary page that you want your user to see. Without the default page, your user will see everything, the folder of your files for projects, activities, and other folders in your server. There's only one default document per folder.

4. Explain the difference between an absolute link and a relative link. Why would you use relative links?

You use relative path if you're in the same domain name. While you use absolute link if you're trying to include an external link or other websites in your webpages. We call it absolute link because it has everything, the domain name, the path, and etc.
