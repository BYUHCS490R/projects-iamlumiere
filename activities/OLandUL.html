3. What are special entity characters?

Special entity characters are a set of characters that are reserved by html that are difficult to type out using your regular keyboard. one example would be a euro sign and a credit sign.

Examples of Special Entity Characters:

< this is a less than

> this is a greater than

  this mean no break space

5. What is the difference between OL and UL Tags?

< ul > is an unordered list tag. it is used to create a list of items inside where the order does not matter and is physically displayed with bullet points. TL;DR - It's an unordered list. Meaning it's ok if its not organize. While the < ol > the opposite of ul tag which is used to create a define list where the order of items is important and explicitly marked with numbers or letters. You can add an element and change a value to it. example would be the type="I". The value of the type changes what are on lettering, number, or the numeral on the left side of the text.

Example: Unordered List

  • Patty
  • Lettuce
  • Tomato
  • Bun
  • Mayo

Example: Ordered List

  1. Bun
  2. Lettuce
  3. Mayo
  4. Patty
  5. Tomato

6. What is a DL?

DL is a decription list or also called a definition list. similar to ol and ul, it is used to list terms with thier definitions and descriptions. With this, you are creating a structure similar to a glossary.

Example: Description List
