Questions

1. Why do we need a viewport tag?

Without the viewport meta tag, it will be messy once you're browsing the page created in your phone. Because we have this tag, it makes it possible to adjust the display.

2. When you select the same tag twice, and in one of them you say "flex-direction: column;" and the other says "flex-direction: row;", which one takes priority?

The one which will takes priotirty would the last one on your code.

3. How do you change the CSS for a web page displayed on a phone?

using @media ONLY screen AND (*the width of your phone) makes it possible to change what your website looks like on your phone.
