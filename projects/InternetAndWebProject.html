What is the Internet?

Is a network of computers and electronic devices around the world. The Internet makes it possible to access information, communicate with everyone through electronic devices, and other things like streaming and watching videos.

What is Web?

Web or the “World Wide Web”, are the websites available to be viewed using web browsers and can only be accessed through the internet.

What is the difference between the internet and the Web?

The internet is the physical architecture of the connection of electronic devices such as computers, satellites, and mobile phones around the world. While the web or the World Wide Web refers to the thousands of information available that is accessed through the internet using web browsers.

What is ASCII and Unicode?

ASCII or American Standard Code for Information Interchange is a limited set of computer encoding that can only write down the basic English letters and characters. This would mean that it can't handle chinese characters or emojis. Unicode is an encoding standard that recognizes different languages and known characters.

What is UTF-8?

The UTF-8 or Unicode Transformation Format - 8 bit is the most character encoding system that represents every character in the unicode standard. In HTML, it makes it possible to use other character or emoji that aren’t available in the ASCII.

What is URL?

The URL or Uniform Resource Locator is the complete address of the information you want to access on the web. One of example of it would be http://google.com/what-is-a-game

What is Domain name?

The domain name is the friendly web address that makes other people familiarize the name of the website. Instead of typing out the IP address, they can just put the specific name like google and type out (.com).

What is HTTP?

HTTP or the Hypertext Transfer Protocol is a protocol that allows end devices to make a request and receive resources like webpages, images, and videos from the servers in the web.

What is HTTPS?

HTTP or the Hypertext Transfer Protocol Secure is basically the secure version of HTTP.

What is IP Address?

IP address or the Internet Protocol Address is the unique address given to an electronic device such as computers, servers, and phones that helps them to be Identified in a network. Also, this address allows it to communicate with other devices connected to the network.

What is DNS?

DNS or Domain Name System is a unique feature that converts a friendly domain name like “google” into its computer readable address like “192.168.0.1”.

What is W3C Recommendations?

The W3C recommendations is the standard or guidelines created by the World Wide Web Consortium to ensure consistency, accountability, and quality.

What is XML, XHTML, HTML, and HTML5?

- XML or Extensible Markup Language is a language that’s responsible for defining the rules and formatting of encoding documents that is both readable by both humans and a machine.
- XHTML or Extensible Hypertext Markup Language is another language that is similar to HTML yet uses the rules designed by XML.
- HTML or the Hypertext Markup Language is coding languages used in developing webpages.
- HTML5 or Hypertext Markup Language 5 is the latest version of HTML that is designed to enhance web development and user experience.
