What is a relative link and absolute link?

A relative link means a way to tell the browser where it can find another file such as webpages, images, and etc that is related to the current file. In addition, instead of providing the whole web address you will only need to provide the path of the files current location or the name of the page/file itself. On the other hand, absolute link means you have to provide a complete URL that includes the protocol, the domain name, and the specific path to a file or a website. In short, using absolute links makes sure that when a user click a link, it make sures that they reach the correct destination.
